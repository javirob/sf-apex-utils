# Salesforce Apex Utilities (Practice Project)

This is a small Salesforce DX practice project created to improve my Apex development skills and learn a professional end-to-end workflow:
- Creating a Salesforce DX project from scratch  
- Developing Apex classes  
- Writing unit tests  
- Deploying to scratch orgs using the Salesforce CLI  
- Managing source control with Git and GitHub  

### Included in this project
- **MyMathUtil** — simple Apex utility class  
- **MyMathUtilTest** — unit tests validating its behavior  

### Running the tests
```bash
sf apex test run -n MyMathUtilTest -w 10
```
